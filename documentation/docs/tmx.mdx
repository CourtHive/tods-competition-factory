## Quick Start

### For End Users

Visit the [live demo](https://courthive.github.io/TMX) and explore tournament management features.

### For Developers

Clone and run locally:

```bash
git clone https://github.com/CourtHive/TMX
cd TMX
npm install
npm run dev
```

### For Documentation Readers

Start with the [TMX Overview](./tmx/overview.md) to understand the architecture, then explore specific features based on your interests.
