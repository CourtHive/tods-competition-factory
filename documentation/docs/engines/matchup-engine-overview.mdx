---
name: MatchUp Engine Overview
title: Overview
---

**matchUpEngine** provides methods for querying against and reporting on `matchUps` along with utilities for parsing/validating `matchUpFormat` codes,
and manipulating `tieFormat` and `score` objects.

```js
import { matchUpEngine } from 'tods-competition-factory';
// -- or --
const { matchUpEngine } = require('tods-competition-factory');
```
