---
title: The Forge
---

Factory methods are exported directly via Generate, Transform and Query packages:

```js
import * as generate from 'tods-competition-factory/generate';
import * as transform from 'tods-competition-factory/transform';
import * as query from 'tods-competition-factory/query';

--or--

const generate = import('tods-competition-factory/generate');
const transform = import('tods-competition-factory/transform');
const query = import('tods-competition-factory/query');
```
