## Credits and Contributors

The **Competition Factory** represents years of development, collaboration, and real-world testing across numerous tournaments and sports organizations. We are grateful to all contributors, testers, and organizations that have provided feedback and support throughout the project's evolution.

### Interactive Documentation

This documentation includes interactive code examples that allow you to explore the Competition Factory APIs directly within your browser:

```jsx live

  const thanks = tournamentEngine.credits();

  return <pre>{thanks}</pre>;

```

### Open Source

The Competition Factory is built on the [Tennis Open Data Standards (TODS)](https://itftennis.atlassian.net/wiki/spaces/TODS/overview), an initiative to create vendor-independent, platform-agnostic data standards for tennis and racquet sports competitions.
