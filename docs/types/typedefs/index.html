                                            matchUp​

                                            A matchUp is an object which contains the details of an encounter between two tournament participants, which can be INDIVIDUALS, PAIRS, or TEAMS. The term matchUp was chosen to differentiate from other interpretations, particularly in the context of software development, of the more traditional term "match". Other descriptive terms such as "face off" and "throw down" were never seriously considered.
